# 📈 Sales Prediction App

A simple machine learning web app that predicts sales based on input features. Built using Python and deployed with Streamlit.

---

## 🚀 Project Overview

This project uses a trained machine learning model to predict sales. Users can enter input values through a web interface and get instant predictions.

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy
* VS Code

---

## 📊 Features

* Easy-to-use web interface
* Real-time sales prediction
* Clean and interactive UI using Streamlit

---

## ▶️ How to Run Locally

1. Clone the repository:

```bash
git clone <your-repo-link>
cd <your-folder-name>
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the app:

```bash
streamlit run app.py
```
